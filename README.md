# AI Narrative Intelligence

Live Demo:
https://2026-nlp-raise-elmt8eifyab9huiuoztk3j.streamlit.app/

An NLP system that transforms 10,500+ AI news headlines into structured behavioral signals across 12 economic, cognitive, and social categories.

## Key Results

| Model | Micro-F1 |
|---------|---------|
| TF-IDF + Logistic Regression | **0.943** |
| DistilBERT | 0.921 |

### Highlights

- Built a multi-label NLP classification pipeline
- Analyzed 10,500+ AI-related news headlines
- Developed a 12-category behavioral taxonomy
- Applied NMF topic modeling for narrative discovery
- Compared narrative framing across Llama, Mistral, and Qwen
- Deployed an interactive Streamlit dashboard
